# FFXIV Rhela Vahl Notes

This repo is a personal Final Fantasy XIV tracking repo for **Rhela Vahl**.

It is not a public guide, not a wiki replacement, and not meant to be authoritative. It is mostly a working handoff/scratchpad system for keeping ChatGPT project context sane between chats.

---

## What this repo is for now

Primary uses:

- Track Rhela's current MSQ lane without needing to restate everything every chat.
- Keep durable gameplay preferences and spoiler boundaries in one place.
- Store small side-project notes separately instead of bloating the main handoff.
- Track limited-time seasonal events that should not get forgotten.
- Track long-term optional goals like titles, glamour, side jobs, retainers, gathering, and market experiments.

The core file is:

- [`rhela_progress.md`](rhela_progress.md) — compact current-state handoff for new FFXIV chats.

That file should stay lean. Detailed side projects belong in separate Markdown files.

---

## Current file pattern

Typical file roles:

- `rhela_progress.md` — main status handoff only.
- `rhela_*_training.md` — job-specific training notes.
- `rhela_professional_title_tracker.md` — progress toward **The Professional** title.
- `seasonal_event_*.md` — time-limited event reminders and reward notes.
- Other focused Markdown files — temporary or durable notes for one specific system or goal.

---

## Update rules

- Do not treat fast-changing values as permanent truth.
- Prefer broad state over exact counts unless an exact update was requested.
- Replace stale notes instead of growing files forever.
- Do not update `rhela_progress.md` unless Kain explicitly asks for it.
- Put narrow project notes in their own file when possible.
- Keep spoiler notes aligned to the current MSQ point.

Fast-changing examples that should not be over-pinned:

- current gil
- exact material counts
- exact retainer inventories
- exact class/job levels unless doing a tracker update pass
- current market prices

## Reliability warning

Some older files in this repo were generated with AI help and may contain bad mechanics, stale advice, or confident nonsense. Treat older duty guides and tactical notes as untrusted unless they have been verified against current FFXIV sources or actual play.

This repo is for personal continuity, not correctness guarantees.

## Current advice style

When using this repo as context, advice should be:

- blunt
- practical
- spoiler-light
- MSQ-first unless another priority is explicitly stated
- hostile to pointless chore lists
- careful about recommending gil spending

The main operational idea:

> Push MSQ, keep retainers moving, use maps/mining/leves/roulettes as useful breaks, and keep side systems from hijacking the lane.
