# FFXIV MSQ Poetics Gear Ladder

Practical gearing note for running MSQ without thinking too hard every few levels.

Scope: combat jobs, especially Warrior / tank MSQ progression. This is not savage gearing, BiS planning, materia optimization, or spreadsheet monk behavior. It is the lazy-correct leveling ladder.

## Core idea

Every old expansion cap has a strong augmented tomestone set bought with **Allagan Tomestones of Poetics**.

Use that set at each `x0` level, ride it through the next expansion, replace pieces naturally when dungeon/MSQ gear beats it, then repeat at the next cap.

The loop:

```text
Hit x0 / unlock vendor
→ buy augmented Poetics weapon + armor
→ ignore gear for a while
→ replace only obvious upgrades
→ repeat at next x0
```

Stay near the **2,000 Poetics cap** before the next vendor unlock so you can buy the important pieces immediately.

## Buy priority

For one main job:

1. **Weapon**
2. **Chest + legs**
3. **Head + hands + feet**
4. **Accessories**
5. **Second ring cleanup / leftovers**

Why:

- Weapon damage matters a lot.
- Chest and legs are huge stat chunks.
- Accessories are useful, but not worth delaying weapon/core armor.
- Tome rings are often `Unique`, so you may not be able to wear two identical augmented rings. Use the best second ring you naturally get from MSQ/dungeons/vendors. Do not lose sleep over this.

For Warrior, buy the **Fending** set.

## Replacement rule while leveling

Do not farm replacement gear while MSQ leveling unless the game blocks you with an item-level requirement.

Use this rule:

```text
If a dungeon/MSQ reward is higher item level, equip it.
If it is not higher item level, ignore it unless the stats are wildly better.
If it is a weapon upgrade, care more.
```

Poetics gear is strongest right at the old expansion cap. Dungeon/MSQ gear usually starts catching up somewhere around the middle/back half of the next expansion.

That is fine. The point is not to be perfectly geared. The point is to avoid gear panic.

## Poetics cap strategy

Since Poetics cap at 2,000:

- Sit around **1,500–2,000** before the next vendor unlock.
- When the set unlocks, buy weapon/chest/legs first.
- Run roulettes/MSQ/dailies to fill out the rest.
- If capped early, buy:
  - alt-job gear for likely future jobs,
  - glamour pieces,
  - GC seal conversion items,
  - relic materials only if you know you need them.

Do not let perfect efficiency turn into teleport-fee clown math.

## Warrior-specific lazy rules

For WAR during MSQ:

- Buy **Fending** gear.
- Prioritize the axe.
- Higher item level beats substat nitpicking while leveling.
- Do not worry about Tenacity vs Skill Speed unless you are doing current endgame optimization.
- If survivability feels bad, check weapon/chest/legs first.
- If threat feels weird, it is probably not the gear unless your weapon is ancient.

## Practical Rhela checklist

At each cap:

```text
[ ] Unlock relevant tomestone vendor
[ ] Buy WAR/Fending weapon
[ ] Buy Fending chest
[ ] Buy Fending legs
[ ] Buy remaining left-side armor
[ ] Buy accessories
[ ] Fill second ring slot with best available
[ ] Stop caring until the next x0 cap
```
