# FFXIV Adventurer Squadron Notes

Quick notes for getting through Grand Company squadron progression without turning it into a second job.

This is focused on two different systems that look related but behave differently:

- **Squadron Missions**: send 4 recruits away for 18 hours. These care about squad stats: **Physical / Mental / Tactical**.
- **Command Missions**: run a dungeon with 3 squad members. These care about party role composition and NPC combat behavior.

## TL;DR

Do not fire recruits just because the roster is awkward early.

Short-term plan:

1. Keep leveling squad members until at least one hits **40**.
2. Complete **Flagged Mission: Crystal Recovery** to unlock **First Lieutenant**.
3. After First Lieutenant, buy class-change books.
4. Convert one spare DPS into **Conjurer** if healer/Mental coverage is weak.
5. Run **5 different Command Missions**.
6. Complete **Flagged Mission: Sapper Strike** to reach **Captain**.

Lazy good roster:

```text
2x Marauder
2x Conjurer
2x Archer
2x Pugilist
```

More optimized mixed roster:

```text
Marauder
Gladiator
Conjurer
Thaumaturge
Arcanist
Archer
Rogue
Pugilist
```

Class to cut if needed:

```text
Lancer
```

## Progression Gates

### Routine Missions

Requires:

```text
At least one squad member level 20
→ Flagged Mission: Voidsent Elimination
→ unlock Routine Missions
```

### First Lieutenant

Requires:

```text
At least one squad member level 40
→ Flagged Mission: Crystal Recovery
→ unlock Priority Missions
→ promote to First Lieutenant
```

This is the current wall if the squad is still under 40.

### Captain

Requires:

```text
Complete 5 different Command Mission dungeons
→ Flagged Mission: Sapper Strike
→ promote to Captain
```

The 5 Command Missions need to be different dungeons. Spamming the same one five times does not satisfy this gate.

## Squadron Missions vs Command Missions

### Squadron Missions

These are the 18-hour independent deployments.

They use:

```text
Physical
Mental
Tactical
```

Your party role composition does **not** matter. You can send four DPS, four tanks, whatever. The mission only cares whether the squad's combined stats meet the requirement.

If all three stat requirements are met, success is guaranteed.

If one or more stats are short, the mission can fail. Failed missions still give partial XP, so sending the squad on a sketchy high-XP mission can still be worth it when you are just trying to level them.

Use these for:

- passive squad XP
- flagged mission progression
- priority mission rewards later
- chemistry bonus rewards

### Command Missions

These are actual dungeons with squad NPCs.

They require a normal light party:

```text
1 tank
1 healer
2 DPS
```

Your own role counts.

Examples:

```text
If you are WAR:
- You fill tank
- Bring 1 Conjurer
- Bring 2 DPS

If you are DPS:
- Bring 1 Gladiator/Marauder
- Bring 1 Conjurer
- Bring 1 DPS

If you are healer:
- Bring 1 Gladiator/Marauder
- Bring 2 DPS
```

Use these for:

- active squad XP grinding
- leveling squad battle tactics
- the 5-different-dungeon Captain requirement
- dungeon loot for Expert Delivery / seals
- alt-job leveling if you want NPC dungeon runs

## Squadron Classes

Squad members only use base classes, not full jobs.

So:

- Marauder, not Warrior
- Gladiator, not Paladin
- Conjurer, not White Mage
- Arcanist, not Summoner/Scholar
- etc.

This matters because their ability kits are weird old fossil kits. Some classes perform better than you would expect because squadron AI still uses older abilities.

## Stat Profiles for Squadron Missions

### Physical-heavy

Good for missions needing Physical.

#### Marauder

Pros:

- Best Physical stat.
- Strong independent mission stat anchor.
- Works as a Command Mission tank.
- Good default tank pick if you do not care about perfect AI behavior.

Cons:

- Bad Mental.
- Only moderate Tactical.
- In Command Missions, tank AI can tunnel vision targets.
- Cone/positioning nonsense may be less idiot-proof than Gladiator.

Use when:

- You need Physical.
- You need a tank.
- You want a simple stat anchor.

#### Gladiator

Pros:

- Strong Physical.
- Better Tactical than Marauder.
- Works as a Command Mission tank.
- AI tanking can feel cleaner because Gladiator has more straightforward AoE enmity behavior.

Cons:

- Less Physical than Marauder.
- Bad Mental.
- Still has normal squadron tank AI stupidity.

Use when:

- You need a tank that is slightly more balanced than Marauder.
- You are building a mixed roster.

#### Pugilist

Pros:

- Balanced Physical/Tactical.
- Useful filler class.
- Helps smooth awkward stat requirements.
- Good in lazy rosters.

Cons:

- Not the best at any single stat.
- Not a top Command Mission DPS pick compared to Arcanist/Archer.

Use when:

- You need flexible stat padding.
- You do not want to over-optimize.

## Mental-heavy

Good for missions needing Mental.

#### Conjurer

Pros:

- Best Mental stat.
- Only actual healer option for Command Missions.
- Mandatory if you want to run Command Missions while playing tank or DPS.

Cons:

- Bad Physical.
- Bad Tactical.
- Healer AI can react slowly.
- You may need to keep pulls sane.

Use when:

- You need Mental.
- You need a healer.
- You are fixing a squad with too much DPS and not enough magic coverage.

#### Thaumaturge

Pros:

- Strong Mental.
- Better Tactical than Conjurer.
- Good independent mission stat piece.

Cons:

- Bad Physical.
- Not a preferred Command Mission DPS.
- Arcanist usually brings more useful dungeon behavior.

Use when:

- You need Mental but do not need another Conjurer.
- You are building the optimized mixed roster.

#### Arcanist

Pros:

- Good Mental/Tactical balance.
- Excellent Command Mission DPS.
- Uses useful old-school squadron abilities like AoE damage.
- Can occasionally help with healing.
- Great pick for fast Command Mission clears.

Cons:

- Low Physical.
- Less raw Mental than Conjurer/Thaumaturge.
- Still AI-controlled, so do not expect genius.

Use when:

- You want a strong Command Mission DPS.
- You want Mental/Tactical coverage.
- You want a DPS that helps make dungeon packs die faster.

## Tactical-heavy

Good for missions needing Tactical.

#### Archer

Pros:

- Best Tactical stat.
- Strong Command Mission DPS.
- Ranged AI is generally less stupid than melee AI.
- Good AoE behavior.

Cons:

- Bad Physical.
- Bad Mental.
- Can pull/engage awkwardly like any squadron NPC.

Use when:

- You need Tactical.
- You want a safe, strong dungeon DPS.
- You are building the lazy roster.

#### Rogue

Pros:

- Strong Tactical.
- Better Physical than Archer.
- Good mixed-roster Tactical piece.
- Solid single-target DPS behavior.

Cons:

- Not as good at AoE pack clearing as Arcanist/Archer.
- Melee AI can waste time moving.

Use when:

- You want Tactical with more Physical than Archer.
- You are using the optimized mixed roster.

#### Lancer

Pros:

- Good Tactical.
- Better Physical than Archer/Rogue in some stat spreads.
- Fine if you already have one leveled and need stats.

Cons:

- Easiest class to cut from optimized rosters.
- Poorer Command Mission value because squad Lancer lacks job-only AoE tools.
- Usually worse than Archer/Arcanist for dungeon speed.

Use when:

- You already have one leveled.
- You need its exact stat spread.
- You have no better Tactical option yet.

## Recommended Rosters

### Lazy Roster

```text
2x Marauder
2x Conjurer
2x Archer
2x Pugilist
```

Why this works:

- Marauders cover Physical.
- Conjurers cover Mental and healer role.
- Archers cover Tactical and Command Mission DPS.
- Pugilists smooth mixed requirements.

This is not perfect. It is just low-brain and good enough.

### Mixed Optimized Roster

```text
Marauder
Gladiator
Conjurer
Thaumaturge
Arcanist
Archer
Rogue
Pugilist
```

Why this works:

- Covers all three stats with fewer duplicates.
- Keeps both tank options.
- Keeps Conjurer for healer.
- Keeps Arcanist/Archer for strong Command Mission DPS.
- Keeps Pugilist as flexible filler.
- Cuts Lancer because it is usually the least useful unique class.

### Current Practical Fix

If the roster is DPS-heavy and missing healer/Mental coverage:

```text
After First Lieutenant:
- Buy Contemporary Warfare: Magicks
- Convert one spare DPS into Conjurer
```

Do not convert:

- highest-level member
- member close to 40 if you need Crystal Recovery
- member with good chemistry
- only strong Tactical/Physical anchor

Convert:

- duplicate DPS
- low-value Lancer/Pugilist/Rogue if stats allow
- whoever costs the least pain

## Class Change Books

Class changes use **Contemporary Warfare** books.

Broad categories:

```text
Defense → Gladiator / Marauder
Offense → Pugilist / Archer / Lancer / Rogue
Magicks → Conjurer / Thaumaturge / Arcanist
```

Important notes:

- Class changes become realistically available after **First Lieutenant**.
- Changing class costs the recruit **1 level**.
- That is usually better than firing the recruit.
- Recruit race/appearance/personality do not affect stats once class and level are fixed.

## Command Mission Party Advice

### Best general setup if player is tank

```text
Player WAR/tank
Conjurer
Arcanist
Archer
```

Why:

- You control pulls and positioning.
- Conjurer heals.
- Arcanist and Archer delete packs better than most alternatives.

### Best general setup if player is healer

```text
Gladiator or Marauder
Arcanist
Archer
Player healer
```

Why:

- You control the healing, so healer AI cannot grief you.
- Arcanist/Archer do good damage.
- Tank NPC takes the hits.

### If player is DPS

```text
Gladiator or Marauder
Conjurer
Arcanist or Archer
Player DPS
```

Why:

- You need NPC tank and healer.
- Runs may be slower and more annoying because you do not control the tank or healer directly.

## Battle Tactics

Squad members start with **Independent**.

They can learn and level tactics from Command Missions. This is RNG.

Tactics:

```text
Independent = HP up, damage up, damage taken down
Offensive   = damage up
Defensive   = HP up, damage taken down
Balanced    = HP up, damage up
```

General recommendation:

```text
Use Offensive on basically everyone once leveled.
```

Reason:

- More damage means faster runs.
- Squad members are already unusually durable.
- Offensive also improves healer output indirectly through higher healing performance.
- Dead enemies do no damage. Revolutionary MMO science.

Reasonable exception:

```text
Use Independent or Defensive if someone is actually dying.
```

But if runs are stable, Offensive is usually the play.

## Command Mission Orders

You get an Orders hotbar in Command Missions.

Useful commands:

### Engage

Tells the squad to attack the target.

Use it to:

- focus targets
- tell tank NPC what to pull
- swap squad attention to boss adds
- force them to stop being useless at the wrong enemy

### Disengage

Tells squad to stop fighting and come back.

Use it when:

- boss becomes invulnerable
- boss drops poison/fire/garbage under itself
- squad is standing in stupid
- mechanics require repositioning

Warning:

```text
If you Disengage, they will not help or heal until you Engage/Re-engage.
```

Do not forget them in timeout.

### Execute Limit Break

Squadron LB is **Ungarmax**.

It is strong because it:

- costs 1 LB bar
- does single-target damage
- gives party damage up

Use it early in boss fights unless the boss is about to become invulnerable.

## Leveling Strategy

### Passive leveling

Use Squadron Missions.

```text
Send best available mission
Wait 18 hours
Take XP
Repeat
```

If just leveling, failure is not the end of the world because failed missions still give partial XP.

### Active leveling

Use Command Missions.

```text
Run highest comfortable Command Mission
Bring the members you want to level
Repeat until you hate paperwork
```

Notes:

- Only squad members brought into the Command Mission get the completion XP.
- Command Missions cost company seals.
- Dungeon loot can often be turned back into seals via Expert Delivery.
- Do 5 different Command Missions anyway for Captain.

### Daily Training

Training is useful for:

- adjusting Physical/Mental/Tactical
- giving some XP
- preparing for flagged missions

Do not overthink training until you are trying to meet a specific flagged mission requirement.

## Promotion Target Missions

### Crystal Recovery

Goal:

```text
Unlock First Lieutenant
```

Requires:

```text
One squad member at 40
Strong enough stats for mission
```

Recommended approach:

- Keep leveling until someone hits 40.
- Use training to line up stats.
- Do not class-change your only near-40 member right before this unless absolutely necessary.

### Sapper Strike

Goal:

```text
Unlock Captain
```

Requires:

```text
5 different Command Missions completed
Then clear Flagged Mission: Sapper Strike
```

Recommended approach:

- Run the easiest 5 different Command Mission dungeons.
- Level/tune squad enough to meet Sapper Strike stats.
- Clear it.
- Stop caring about squadron daily chores unless you want manuals or alt leveling.

## After Captain

Squadron becomes less urgent.

Still useful for:

- weekly Priority Missions
- Squadron manuals/buffs
- alt leveling through Command Missions
- chemistry reward junk
- completionist disease

Not really useful for:

- direct gil printing
- core MSQ progression
- replacing Trusts long-term

After Captain, treat it as:

```text
weekly buff printer + optional alt leveling system
```

## Practical Recommendations for Current State

If currently Second Lieutenant and roster is awkward:

```text
Do not fire anyone.
Keep leveling.
Send missions even if success chance is not perfect.
Use Command Missions if you want active squad XP.
Wait until First Lieutenant to fix classes cleanly.
```

Once First Lieutenant:

```text
Buy Magicks book.
Convert duplicate DPS to Conjurer.
Aim roster toward lazy or mixed setup.
Run 5 different Command Missions.
Push Sapper Strike.
Hit Captain.
```
