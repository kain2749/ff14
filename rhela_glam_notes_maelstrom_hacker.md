# FFXIV Glam Notes — Rhela Vahl

Purpose: quick reference for two glamour plates:

1. Maelstrom Officer / Captain look
2. Hacker streetwear look

Character vibe:

- Rhela Vahl
- Female Miqo'te WAR
- Default identity: streetwear axe tank
- Maelstrom-aligned
- Big axe stays funny and correct

---

## 1. Maelstrom Officer / Captain Glam

### Core idea

A formal-but-still-dangerous Maelstrom officer look.

Not full plate. Not generic naval NPC.

More like:

> Maelstrom captain who can sign the paperwork, then personally solve the problem with an axe.

This is the glam to wear after making Captain, probably for a day or two, then swap back to streetwear.

### Recommended pieces

| Slot | Main pick | Notes |
|---|---|---|
| Head | Storm Lieutenant's Tricorne | Optional. Very Maelstrom, but test with Miqo'te ears/hair. |
| Body | Storm Lieutenant's Jacket | Anchor piece. This is the whole reason for the glam. |
| Hands | Lominsan Officer's Gloves | Test with jacket cuffs. Hide with Emperor's New Gloves if cursed. |
| Legs | Best Man's Slacks / Falconer's Bottoms / dark slim pants | Pants should be boring. Jacket does the work. |
| Feet | Lominsan Officer's Boots | Clean official Maelstrom boot option. |
| Weapon | Storm Captain's Labrys | Government-issued axe once Captain. Funny and appropriate. |
| Accessories | Emperor's New where needed | Hide clutter unless it intentionally improves the look. |

### Quartermaster / source notes

Most of the real Maelstrom pieces come from the Storm Quartermaster:

- Location: Limsa Lominsa Upper Decks, around X:13.1 Y:12.7
- Currency: Storm Seals
- Requires Maelstrom rank unlocks

Known costs / requirements:

| Item | Cost | Requirement |
|---|---:|---|
| Storm Lieutenant's Jacket | 21,000 Storm Seals | First Storm Lieutenant |
| Storm Lieutenant's Tricorne | 12,000 Storm Seals | First Storm Lieutenant |
| Lominsan Officer's Gloves | 4,650 Storm Seals | Second Storm Lieutenant |
| Lominsan Officer's Boots | 5,100 Storm Seals | Second Storm Lieutenant |
| Storm Captain's Labrys | 35,000 Storm Seals | Storm Captain |

### Build variants

#### Formal Captain

```text
Storm Lieutenant's Tricorne
Storm Lieutenant's Jacket
Lominsan Officer's Gloves
Best Man's Slacks / dark slim pants
Lominsan Officer's Boots
Storm Captain's Labrys
```

Best if the hat behaves.

#### No-Hat Captain

```text
No headpiece or normal glasses/facewear
Storm Lieutenant's Jacket
Lominsan Officer's Gloves or Emperor's New Gloves
Dark slim pants
Lominsan Officer's Boots
Storm Captain's Labrys
```

Probably safer for Rhela if the tricorne fights the hair/ears.

#### Battle Captain

```text
Storm Lieutenant's Jacket
Darker gloves
Darker boots
Storm Captain's Labrys
Minimal accessories
```

Less parade uniform, more boarding-action officer.

### What to test

- Does the tricorne look good with Miqo'te ears?
- Do Lominsan Officer's Gloves clip or fight the jacket cuffs?
- Do the boots look too formal or just right?
- Does Storm Captain's Labrys look clean enough, or too plain compared to the outfit?
- Does the glam still look like Rhela, or just an NPC uniform?

If something does not belong:

- use Emperor's New Gloves
- hide accessories
- remove hat
- keep jacket + boots + axe as the core

## 2. Hacker Streetwear Glam

### Core idea

Cyber streetwear Rhela.

Same baseline identity:

- hoodie / casual jacket
- jeans
- casual shoes or boots
- glasses/cyber eye
- axe

But with a magitek/cyber twist.

Not full sci-fi armor. Not Allagan robot cosplay.

More like:

> streetwear axe tank with a cyber eye and a weapon that looks like it came out of a bad server room.

### Recommended pieces

| Slot | Main pick | Notes |
|---|---|---|
| Facewear | Scaevan Headgear | Cyber eye / scouter piece. Permanent unlock. Buy from Market Board if cheap. |
| Body | Plain Hooded Tunic / Casual Jacket | Hoodie is the goal. Casual Jacket also works if hoodie is unavailable/too expensive. |
| Hands | Emperor's New Gloves / simple dark gloves | Do not overcomplicate it. |
| Legs | Stonewashed Twill Trousers / other jeans | Keep the Rhela streetwear identity. |
| Feet | Calfskin Rider's Shoes / sneakers / Strife Boots | Casual footwear. Avoid heavy armor boots unless intentionally going darker. |
| Weapon | Augmented Scaevan Magitek Axe | Best direct match for Scaevan Headgear. Cyber/magitek axe. |
| Accessories | Mostly hidden | Cyber eye is already the accessory statement. |

### Hacker look priorities

1. Buy/use `The Faces We Wear - Scaevan Headgear`.
2. Equip Scaevan Headgear as facewear.
3. Use hoodie/casual jacket + jeans.
4. Use a magitek/cyber axe.
5. Keep everything else simple.

### Weapon options

#### Best match

```text
Augmented Scaevan Magitek Axe
```

Why:

- matches Scaevan Headgear
- WAR/MRD usable
- level 70
- item level 400
- glamour dresser compatible
- magitek/cyber look without going full raid glow

#### Other possible vibes

```text
Augmented Ironworks Magitek Axe
```

Older ARR magitek look. More industrial, less polished.

```text
Allagan / Dreadwyrm / Alexander-style axe
```

More ancient-tech or raid-tech. May be too loud depending model.

```text
Current black oversized axe
```

Still works if the cyber axe is not available yet. Black weapon + cyber eye + streetwear is still coherent.

### Build variants

#### Clean Hacker

```text
Scaevan Headgear
Plain Hooded Tunic
Stonewashed Twill Trousers
Calfskin Rider's Shoes
Augmented Scaevan Magitek Axe
Emperor's New Gloves
Hidden accessories
```

Best version if the hoodie works.

#### Street Tech

```text
Scaevan Headgear
Casual Jacket
Stonewashed Twill Trousers
Calfskin Rider's Shoes / Strife Boots
Augmented Scaevan Magitek Axe
```

Keeps the existing off-duty/streetwear feel, just adds cyber hardware.

#### Dark Tech

```text
Scaevan Headgear
Dark hoodie or dark casual jacket
Dark jeans
Dark boots
Augmented Scaevan Magitek Axe
```

More serious. Less casual. Good if the axe looks aggressive.

### What to avoid

- Too much glowing armor.
- Too much Allagan neon unless intentionally going hard cyberpunk.
- Giant fantasy/primal axe with big elemental glow.
- Hats that conflict with Scaevan facewear.
- Jewelry clutter.

The cyber eye should be the visual hook. The axe should support it. The clothes should still say Rhela.
