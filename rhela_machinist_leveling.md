# Rhela Vahl — Machinist Leveling Guide

Purpose: keep a practical Machinist leveling plan separate from `rhela_progress.md`.

This is not the main lane right now. Main lane is still MSQ. Machinist is a possible future XP sponge / streetwear tech-gunner alt if WAR starts getting too close to level cap.

---

## Why Machinist

Machinist fits Rhela pretty well:

- streetwear / hacker / tech-gunner glam potential
- ranged DPS comfort for overworld trash
- starts at level 30 instead of level 1
- uses Aiming gear, which can be handled with Poetics at expansion breakpoints
- good candidate for Wondrous Tails turn-ins once WAR does not need the XP

Downside:

- It does **not** start high like Viper.
- It is not an instant XP sponge.
- If the goal is purely efficiency, Viper is faster because it starts at 80.
- If the goal is style/fun, Machinist is probably more Rhela.

---

## Unlock

Unlock Machinist in Ishgard after reaching Heavensward access.

- Job: **Machinist**
- Starting level: **30**
- Role: **Physical ranged DPS**
- Gear type: **Aiming**

After unlocking, do the job quests as they become available. They matter for skills and basic job flow.

---

## Core rule

Do not derail MSQ unless WAR is actually approaching the level cap problem.

Best use pattern:

1. Keep pushing MSQ on WAR.
2. Feed MCH passive/weekly XP when convenient.
3. If WAR is close to cap, do a deliberate MCH leveling push.
4. Once MCH catches the current MSQ level, use it for quest turn-ins when safe.
5. Continue doing actual hard/comfy work on WAR when needed.

---

## XP sponge rule

MSQ quest XP goes to the job that turns in the quest.

So the trick is:

1. Do combat/dungeons/solo duties as WAR.
2. Swap to MCH before clicking the green-check NPC.
3. Turn in on MCH.
4. Swap back to WAR if the next step has combat.

Caveat:

- MCH must be at least the quest's required level.
- If the quest requires level 81, a level 80 MCH cannot turn it in.

## Leveling plan

### 30–50

Goal: get MCH to the first Poetics breakpoint.

Good options:

- **Palace of the Dead** unlock/climb.
- **Palace of the Dead 51–60 spam** once unlocked.
- **Leveling Roulette** when DPS queue is tolerable.
- **Frontline roulette** if PvP is tolerable.
- **Squadron Command Missions** if available and convenient.
- Hunting log is not relevant for MCH because it starts as a job, not a base class with a normal class hunting log path.

Practical read:

- PotD is the repeatable engine.
- Roulettes are the daily chunks.
- WT is not available every day, but is excellent when ready.

### 50–60

Buy Poetics gear:

- **Augmented Ironworks Aiming** gear at level 50.

XP options:

- Leveling roulette.
- Alliance roulette if queue/time feels okay.
- MSQ roulette if the time investment is acceptable.
- PotD 51–60 spam if you want repeatable progress.
- Wondrous Tails turn-in on MCH when available.

### 60–70

Buy Poetics gear:

- **Augmented Shire Aiming** gear at level 60.

XP options:

- Leveling roulette.
- Alliance roulette.
- Heaven-on-High once unlocked/relevant.
- Wondrous Tails turn-in.
- Highest comfortable dungeon if queue is not awful.

### 70–80

Buy Poetics gear:

- **Augmented Scaevan Aiming** gear at level 70.

XP options:

- Leveling roulette.
- Alliance roulette.
- Wondrous Tails turn-in.
- Highest comfortable dungeon if queue is tolerable.
- Later side systems can help, but do not chase them before the MSQ lane naturally gets there.

### 80+

At this point MCH can start acting as a real MSQ XP sponge if current MSQ quest levels line up.

Gear plan:

- Use expansion-appropriate Poetics gear when available.
- Do not overbuy if MCH is only being used for turn-ins.
- If actually playing MCH in duties, keep the weapon reasonably current.

## Palace of the Dead notes

Palace of the Dead is the main spam option for early MCH leveling.

What matters:

- Normal gear mostly does not matter inside.
- PotD uses its own aetherpool weapon/armor progression.
- Clear 10-floor sets for outside XP.
- The usual leveling spam is **floors 51–60** after unlocking them.

Difficulty:

- Leveling spam is not the hard version of Deep Dungeon.
- The hard version is high-floor pushing / solo title attempts.
- For MCH leveling, it is mostly repeatable dungeon treadmill content.

Rules for not wasting time:

- Do not get greedy with traps/chests if the party is messy.
- Finish the 10-floor set cleanly.
- Expect repetition.

## Wondrous Tails plan

WT is excellent for Machinist because it is a big XP chunk with little actual MCH play required.

Use WT turn-ins on MCH when:

- WAR does not need the XP.
- MCH is behind and needs catch-up.
- MCH is close to the next MSQ level requirement.

Do not overthink sticker optimization for this purpose. The goal is XP, not perfect lines.

---

## Roulette policy

MCH is DPS, so queue times can suck.

Use roulettes when they are worth it:

- Leveling roulette: good daily chunk if queue is okay.
- Frontline roulette: strong XP if PvP is tolerable.
- Alliance roulette: decent if time/queue line up.
- MSQ roulette: big but long; only do it if mentally prepared.

Do not turn this into a daily chore list. The point is catch-up, not a second job.

---

## Poetics gear plan

Use Poetics to avoid marketboard spending.

Breakpoints:

- 50: Augmented Ironworks Aiming
- 60: Augmented Shire Aiming
- 70: Augmented Scaevan Aiming
- 80/90: use the appropriate expansion Poetics aiming sets when unlocked/relevant

Priority if Poetics are limited:

1. Weapon
2. Chest/legs
3. Other left-side armor
4. Accessories

If MCH is only being used for turn-ins, gear can lag a bit. If MCH is doing duties, keep gear sane.

Target vibe: **streetwear hacker gunner**.

Possible look language:

- techwear jacket
- dark or slate-grey palette
- casual shoes/boots
- glasses/visor/facewear if available
- compact firearm if a huge gun looks goofy
- urban contractor / off-duty cyber gunner energy

This should fit Rhela's existing casual streetwear identity instead of feeling like a totally separate character.

## Practical TLDR

Machinist is a fun solution, not the fastest solution.

If WAR gets close to cap and XP waste starts bothering you:

1. Unlock MCH.
2. Push it with PotD + WT + roulettes.
3. Gear it with Poetics at 50/60/70.
4. Use it for MSQ turn-ins once it reaches the quest level.
5. Keep WAR as the comfy job for actual MSQ combat.
